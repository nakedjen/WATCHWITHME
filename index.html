Watch With Me • browser demo with real TMDb data
Taste concierge with standards

A prettier little machine with real movie data.

This version fetches live titles from TMDb, scores them for quality, mood, and taste fit, and keeps a local memory of what you loved, liked, rejected, or would happily launch into the sea.

How to use it

1. Paste a TMDb API Read Access Token or v3 API key.
2. Choose your mood, services, and anti-slop threshold.
3. Ask Watch With Me what to watch tonight.
4. Rate the results so the app learns your taste.
5. Save as demo.html and share it around.
